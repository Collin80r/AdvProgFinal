# AdvProgFinal
## Installation
 1. Install cmake
 2. Get local cmake clone
 ```sh
 git clone https://github.com/libsdl-org/SDL.git vendored/SDL
 ```
 3. Configure & Build
 ```sh
 cmake -S . -B build
 cmake --build build
 ```
 3. Run
 ```sh
 ./build/Debug/main
 ```
